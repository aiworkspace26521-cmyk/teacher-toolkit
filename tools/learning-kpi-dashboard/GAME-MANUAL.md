## 1. 遊戲概述

學習 KPI 管理系統將 Pokemon 遊戲化架構重建為學習 KPI 管理系統，
保留 gamification 核心（等級、經驗值、成就）但以「學習數據驅動」取代「戰鬥模擬」。

### 1.1 學生角色

| 角色 | 說明 |
|------|------|
| **Neil（苡逢）** | 學生 A，五年級 |
| **Emma（苡瑄）** | 學生 B，五年級 |
| **管理員** | 家長/教師，可模擬時間、發放資源 |

### 1.2 核心資源

| 資源 | 說明 | 獲取方式 |
|------|------|---------|
| **EXP（經驗值）** | 提升寶可夢等級 | 每日任務提交、戰鬥勝利 |
| **金幣** | 商城貨幣 | 每日任務提交、戰鬥勝利 |
| **徽章** | 道館挑戰成就 | 道館戰勝利 |
| **道具** | 戰鬥消耗品 | 商城購買 |

## 3. 每日任務與分數系統

### 3.1 任務類別

| 類別 | 科目 | 滿分 | 任務數 |
|------|------|------|-------|
| 📐 數理道館 | 代數、計算 | 30 | 2 |
| 📖 國語道館 | 國學、閱讀 | 25 | 2 |
| 🎧 英文道館 | PET/FCE、閱讀 | 25 | 2 |
| ✨ 紀律與體能 | 運動、小提琴、家務、睡眠 | 20 | 4 |
| **每日滿分** | | **100** | **10** |

### 3.2 分數計算

```
最終分數 = (勾選任務分數總和) × 聯盟倍率

聯盟倍率：
- 一般時期：1x
- 聯盟時期（每月最後3天）：2x
```

### 3.3 提交獎勵（預設模式）

```
EXP   = 分數 × 10
金幣  = 分數 × 2
```

選擇「挑戰新目標」或「強化現有技能」時獎勵公式略有不同（見第 4、7 節）。

### 3.4 時間系統

- 月份計算：以 2026 年 4 月為基準（第 1 月），自動遞增
- 聯盟時期：每月最後 3 天自動偵測
- 管理員可透過下拉選單模擬不同月份

## 4. 捕捉系統（挑戰新目標）

V3.0 修復：原本此功能為純裝飾性按鈕，現已完整實作。

### 4.1 分數門檻與稀有度

| 每日分數 | 最高可捕捉等級 | 稀有度分配 | 初始等級 |
|---------|-------------|-----------|---------|
| 0~49 | ❌ 無法捕捉 | — | — |
| 50~59 | 一般系 | 100% 一般 | Lv.5~7 |
| 60~74 | 一般系 | 85% 一般 / 15% 稀有 | Lv.5~9 |
| 75~94 | 稀有系 | 60% 一般 / 35% 稀有 / 5% 傳說 | Lv.12~15 |
| 95~100 | 稀有/傳說 | 25% 一般 / 60% 稀有 / 15% 傳說 | Lv.23~25 |

### 4.2 屬性選擇

開啟「挑戰新目標」時會顯示屬性下拉選單，支援 18 屬性 + 隨機：

```
🎲 隨機 → 從對應稀有度隨機選取
🔥 火   → 從對應稀有度中篩選火系寶可夢
💧 水   → 篩選水系
🌿 草   → 篩選草系
...（共 18 屬性）
```

### 4.3 捕捉流程

1. 學生勾選任務、選擇「挑戰新目標」
2. 選擇指定屬性（或隨機）
3. 預覽顯示可達到的稀有度等級
4. 提交 → 系統根據分數判定稀有度 + 從屬性池隨機選取
5. 生成寶可夢 ID、等級、雙屬性
6. 事件寫入 Firestore（`action: "捕捉"`）
7. Toast 通知捕捉結果

### 4.4 捕捉獎勵

```
捕捉模式：
EXP   = 分數 × 8（小幅降低，因獲得新寶可夢）
金幣  = 分數 × 1.5
```

### 6.2 雙屬性系統（V3.0 新增）

V3.0 前所有寶可夢僅為單一屬性。V3.0 加入 `POKEMON_SPECIES_TYPES` 資料庫（104 筆記錄），
依據真實寶可夢遊戲設定雙屬性。

**範例**：
- 妙蛙花：草 + 毒
- 噴火龍：火 + 飛行
- 暴飛龍：龍 + 飛行
- 路卡利歐：格鬥 + 鋼

**屬性相剋計算規則**：
```
總相剋倍率 = TYPE_CHART[攻擊屬性][防禦屬性1] × TYPE_CHART[攻擊屬性][防禦屬性2]

範例：水槍攻擊 妙蛙花（草/毒）
  = TYPE_CHART["水"]["草"] × TYPE_CHART["水"]["毒"]
  = 0.5 × 1.0 = 0.5（效果不好）

範例：火系攻擊 妙蛙花（草/毒）
  = TYPE_CHART["火"]["草"] × TYPE_CHART["火"]["毒"]
  = 2.0 × 1.0 = 2.0（效果絕佳）
```

## 7. 養成系統

### 7.1 訓練方式

#### A. 每日任務（均衡成長）

提交任務後所有寶可夢獲得 EXP（透過 `recalculateStudentState` 自動分配）。

#### B. 強化現有技能（集中訓練）— V3.0 修復

選擇「強化現有技能」後，可指定單一寶可夢獲得全部 EXP：

```
提交後 action = "糖果升級"
指定寶可夢獲得 EXP = 分數 × 10
（其他寶可夢僅透過學習裝置獲得 10~20%）
```

#### C. 戰鬥 EXP

戰鬥勝利後 EXP 分配：
- 先發寶可夢：獲得經驗 × 1.2
- 其他參與寶可夢：獲得經驗 × 0.3
- 持有學習裝置時非參與者獲得 20%

#### D. 商城道具

- 神奇糖果（25 幣）：全隊獲得大量 EXP

### 7.2 等級公式

```
當 Lv ≤ 20：升等需求 = Lv × 100
當 21 ≤ Lv ≤ 40：升等需求 = Lv × 150
當 41 ≤ Lv ≤ 60：升等需求 = Lv × 200
當 61 ≤ Lv ≤ 80：升等需求 = Lv × 300
當 81 ≤ Lv ≤ 99：升等需求 = Lv × 500

溢滿 EXP 自動循環至下一級
最高等級：Lv.99
```

### 7.3 種族值公式

```
HP    = 40 + Lv × 3
攻擊  = 30 + Lv × 2
防禦  = 30 + Lv × 2
特攻  = 30 + Lv × 2
特防  = 30 + Lv × 2
速度  = 30 + Lv × 2
```

**傳說加成**：
- 一般傳說：HP×1.2, 攻擊×1.3, 防禦×1.2, 特攻×1.3, 特防×1.2, 速度×1.15
- 超級傳說（超夢、烈空坐等）：HP×1.35, 攻擊×1.45, 防禦×1.35, 特攻×1.5, 特防×1.35, 速度×1.3

**最終進化加成**（非傳說）：
- 攻擊×1.1, 防禦×1.05

## 9. 戰鬥系統

### 9.1 戰鬥流程

```
回合制輪流攻擊（速度高者先攻）
  │
  ├─ 狀態檢查（凍結/睡眠/麻痺）
  ├─ 招式選擇（4 選 1）
  ├─ 傷害計算
  ├─ 狀態/異常判定
  ├─ 燒傷/寄生種子持續傷害
  └─ 切換下一隻（若當前昏厥）
```

### 9.2 傷害公式

```
物理招式：
  實際攻擊 = 攻擊方攻擊 × 1.1^(攻階)
  實際防禦 = 防禦方防禦 × 1.1^(防階)
  基礎傷害 = floor( ((等級 × 2/5 + 2) × 威力 × 實際攻擊 / 實際防禦) / 50 + 2 )

特殊招式：
  實際特攻 = 攻擊方特攻 × 1.1^(特攻階)
  實際特防 = 防禦方特防 × 1.1^(特防階)
  基礎傷害 = floor( ((等級 × 2/5 + 2) × 威力 × 實際特攻 / 實際特防) / 50 + 2 )

修正項目（依序疊乘）：
  1. 傳奇 vs 傳奇加成：×1.5（雙方均為傳說時）
  2. 屬性相剋：×0/×0.5/×1/×2
  3. 最小值保護：max(1, 傷害)
  4. 隨機浮動：×0.85~1.0
  5. 會心一擊（6.25%）：×1.5
  6. 最終傷害：min(計算值, 目標剩餘 HP)
```

### 9.3 速度判定

```
誰先攻 = 比較雙方速度 × 1.1^(速階)
速度高者先攻，每回合輪流
```

### 9.4 狀態異常

| 異常 | 觸發率 | 效果 |
|------|-------|------|
| 🔥 燒傷 | 30%（火系招式） | 每回合損失 maxHP × 6.25% |
| ⚡ 麻痺 | 30%（電系招式） | 25% 機率無法行動 |
| ❄️ 凍結 | 20%（冰系招式） | 完全無法行動 |
| ☠️ 中毒 | 30%（毒系招式） | 每回合持續傷害 |
| 💤 睡眠 | 60%（超能招式） | 50% 機率每回合甦醒 |
| 🌱 寄生種子 | 特殊招式 | 每回合吸取 maxHP × 8.33% |

### 9.5 能力變化

```
每個階段 = ×1.1^n
範圍：-6（最低）~ +6（最高）

提升招式：劍舞(+2攻)、詭計(+2特攻)、龍之舞(+1攻+1速)
           鐵壁(+2防)、冥想(+1特攻+1特防)
降低招式：叫聲(-1攻)、搖尾巴(-1防)、潑沙(-1命中)
```

### 9.6 招式分類

102 個招式分為三大類：

| 分類 | 說明 | 範例 |
|------|------|------|
| 物理 | 吃攻擊 vs 防禦 | 地震、近身戰、咬碎 |
| 特殊 | 吃特攻 vs 特防 | 大字爆、水炮、龍星群 |
| 變化 | 不直接造成傷害 | 劍舞、守護、催眠術 |

## 13. PvP 對戰系統

### 13.1 開啟條件

- 隊伍中至少 2 隻寶可夢
- 透過「宿敵」按鈕開啟

### 13.2 對戰流程

1. **選擇上場寶可夢**：從隊伍中挑選 1 隻
2. **AI 選擇對手**：隨機選擇隊伍中不同 ID 的寶可夢
3. **回合制對戰**：
   - 速度高者先攻
   - 玩家選擇 4 個招式之一
   - AI 隨機選擇招式
   - 雙方輪流攻擊直到一方 HP 歸零

