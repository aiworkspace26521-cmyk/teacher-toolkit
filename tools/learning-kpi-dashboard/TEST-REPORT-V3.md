### EXP 成長曲線計算

**核心公式** `calcLevelAndExp(totalExp, initialLevel=5)`:
- Level N needs `N × 100` EXP to gain 1 level
- `sum(5..L) × 100 = EXP_needed_for_level_L`
- `50 × (L² + L - 20) = total_EXP`

#### 5. EXP 曲線重設計

**當前問題**：線性需求（`L × 100`）導致前期升級太快、後期升級太慢。

**建議方案**（階梯式曲線）：
```javascript
// Level 5~20: L × 100（新手期快速成長）
// Level 21~40: L × 150（穩定期）
// Level 41~60: L × 200（挑戰期）
// Level 61~80: L × 300（高手期）
// Level 81~99: L × 500（大師期）
function calcLevelAndExp(totalExp, initialLevel) {
  var lvl = initialLevel || 5, cur = totalExp;
  var expNeeded = getExpNeeded(lvl);
  while (cur >= expNeeded) {
    cur -= expNeeded;
    lvl++;
    expNeeded = getExpNeeded(lvl);
    if (lvl >= 99) { lvl = 99; break; }
  }
  ...
}
function getExpNeeded(lvl) {
  if (lvl <= 20) return lvl * 100;
  if (lvl <= 40) return lvl * 150;
  if (lvl <= 60) return lvl * 200;
  if (lvl <= 80) return lvl * 300;
  return lvl * 500;
}
```
