## P0：立即修復（估時合計 ~3h）

### P0-1：雙屬性系統（~2h）
- **問題**：`getPokemonType()` 只取第一組括號，妙蛙花應為「草/毒」但只回「草」
- **範圍**：`SPECIES_TYPES` 已存雙屬性（如 `"妙蛙花": "草/毒"`），但 parser 未解析
- **修改**：
  1. `getPokemonType()` → 回傳陣列 `["草","毒"]`
  2. `TYPE_CHART` 相剋計算 → 雙屬性相乘（草 x 毒 = 4x 超能、0.25x 草）
  3. 戰鬥 UI 顯示雙屬性圖示
  4. `getEffectiveness()` 改為計算雙屬性
- **相依**：P1-2 可合併實作

### P0-2：伊布進化後招式池刷新（~0.5h）
- **問題**：伊布進化後 `getPokemonMoveset()` 仍回一般系招式
- **修改**：`recalculateStudentState()` 進化流程中更新寶可夢的 `baseName`，`getPokemonMoveset()` 根據進化後型態給正確招式

### P0-3：戰中道具消耗寫入 Firestore（~0.5h）
- **問題**：`useItemInArena()` 只改 `globalData`，存檔時未寫入資料庫
- **修改**：道具使用後將消耗事件寫入 `kpi_events`，確保 reload 後道具量正確
