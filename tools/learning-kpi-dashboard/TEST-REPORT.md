### 發現的潛在優化點（非 bug）

1. **前端載入狀態指示** — 當 Firebase SDK 初始化較慢時，可考慮更明確的載入動畫
2. **批次操作** — Admin 每次只能 +500 幣，可考慮自訂數量輸入
3. **離線緩存** — 目前完全依賴 Firebase 在線，可考慮加 localStorage 快取
