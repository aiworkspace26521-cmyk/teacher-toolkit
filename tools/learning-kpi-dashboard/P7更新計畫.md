### P7-1：路人戰 Gen2~9 整合（2h）

**現狀**：POKEMON_TIERS + pokemon-gen2-9.js 已有 501 種 Gen2~9 寶可夢，但 `generateEnemy()` 中缺少 `getRegionPool()` 函數，路人戰對手從全池隨機選取，無地區分佈邏輯。

**工作項目**：

```js
// 新增 getRegionPool(region) — 依地區過濾可出現的寶可夢
function getRegionPool(region) {
  var regionPools = {
    "關都": ["波波","烈雀","小拉達",...],  // Gen1 為主
    "城都": ["咕咕","尾立","土狼犬",...],  // Gen2 為主
    "合眾": ["小貓怪","姆克兒",...],       // Gen4 為主
    "卡洛斯": [...],                       // Gen6 為主
  };
  return regionPools[region] || regionPools["關都"];
}
```

1. 建立 4 地區 × 各 ~50 種常見寶可夢的區域池（從 POKEMON_TIERS 中選取）
2. 依玩家徽章數決定當前地區（8 徽=關都、16 徽=城都、24 徽=合眾、32 徽=卡洛斯）
3. 路人戰對手等級依地區遞增（關都 lvBonus -5、城都 ±0、合眾 +5、卡洛斯 +10）
4. 在戰鬥標題顯示地區標記

**影響檔案**：`frontend/kpi-dashboard.html`（新增 `getRegionPool()` + 修改 `generateEnemy()`）

**現狀**：4 種天氣（雨/晴/沙暴/冰雹）均已實作，天氣粒子效果 `spawnWeatherParticles()` 存在。但缺乏招式與天氣的互動（陽光烈焰免充能）。

**工作項目**：

1. **陽光烈焰/日光刃晴天免充**：在 `executeTurn()` 中，若招式為「陽光烈焰」或「日光刃」且天氣為「大晴天」，跳過充能回合直接攻擊並 +1.5x 傷害

   - 大雨：增加藍色粒子 + 螢幕震動
   - 晴天下火系招式增加火焰殘影

**影響檔案**：`frontend/kpi-dashboard.html`（`executeTurn()`、`spawnWeatherParticles()`、`MOVE_DATABASE`）
