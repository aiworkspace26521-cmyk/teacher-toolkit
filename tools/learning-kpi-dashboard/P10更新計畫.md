## P10-1：雙屬性系統（~2h）

**問題**：目前 `getPokemonType()` 只取第一組括號，眾多雙屬性寶可夢（妙蛙花草/毒、暴鯉龍水/飛行、耿鬼幽靈/毒、烈咬陸鯊龍/地面…）在系統中僅為單一屬性，導致屬性相剋計算不準確。

- [x] 修改 `getPokemonType()` 支援回傳屬性陣列（如 `["草","毒"]`）
- [x] 修改 `TYPE_CHART` 有效性計算 → 雙屬性相乘（4x/2x/0.5x/0.25x/0x）
- [x] `POKEMON_SPECIES_TYPES` 補上完整雙屬性（262 種物種對應真實遊戲）
- [x] 戰鬥中屬性相剋提示顯示雙屬性訊息
- [x] PM 詳細資訊面板顯示雙屬性徽章

**影響檔案**：`frontend/kpi-dashboard.html`、`public/kpi-dashboard.html`、`simulation-test.js`
